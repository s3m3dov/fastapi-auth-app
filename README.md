## Initial Set-up ##
### Run using docker-compose ###
* Make sure docker is running on your hardware
```
docker-compose up -d --build
```
or
```
docker-compose -f docker-compose-local.yml up -d --build
```
* Go to [localhost](http://localhost:8000)
